## Tools

### [stm32-mcp](./stm32-mcp/)

**Give Claude Code live, read/write access to a running STM32 firmware — no printf, no recompile.**

An MCP (Model Context Protocol) server that connects Claude Code to an STM32 debug session via `arm-none-eabi-gdb` in MI mode. Read variables by DWARF name, inspect state machines, dump memory, inject values, set breakpoints — while the firmware keeps running.

→ [Full documentation](./stm32-mcp/README.md)
